# Hogwarts Library

A Hogwarts Library é uma landing page interativa que consome uma API contendo informações detalhadas sobre os personagens do universo de Harry Potter. A plataforma permite explorar os personagens de forma dinâmica, com recursos visuais e interativos que tornam a experiência mais imersiva para fãs da saga.

## 📌 Próximos passos:

- Expandir o uso de animações e efeitos visuais para tornar a navegação mais envolvente.
- Otimizar a performance de carregamento dos backgrounds na página inicial, garantindo uma experiência mais fluida.
- Implementar filtros de busca avançados por nome e casa dos personagens, facilitando a pesquisa e exploração.
- Criar componentes interativos de feitiços, trazendo elementos do mundo mágico para a interface.
- Aplicar melhorias de tipagem e padronização de código usando o ESLint, aumentando a qualidade e consistência do projeto.

## ▶️ Como Executar

1. **Clone o repositório:**

```bash
git clone https://github.com/rafasp97/Hogwarts-Library.git
```

2. **Instale as dependências:**

```bash
npm install
```

3. **Rode o projeto:**

```bash
npm run dev
```
